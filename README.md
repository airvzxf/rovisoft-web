# Website | RoviSoft

**RoviSoft.net** is not a traditional website: it's an **interactive command-line terminal** (TTY) that functions as a personal homepage. Visitors arrive and find themselves in a simulated UNIX terminal, where the mouse is practically useless and everything is navigated by typing commands. The aesthetics are inspired by classic terminal emulators: dark background, monospaced font, and _blinking cursor_.
